# Code of Conduct

## The short version

Be respectful. Be direct. Keep it about the code.

## Guidelines

1. **Focus on the work.** Discussions should be about the project — bugs, features, design decisions, code quality. This is a software project, not a platform for unrelated debates.
2. **Be respectful.** Disagree with ideas, not people. No personal attacks, no harassment, no trolling.
3. **Be direct.** Say what you mean. Give honest feedback. Accept honest feedback.
4. **Help others.** If someone is stuck, point them in the right direction.

## Enforcement

Violations can be reported to **conduct@chagg.dev**. Reports are reviewed by the maintainer. Consequences range from a warning to a permanent ban, depending on severity.
